# Contributing to this eBook Collection

Thank you for your interest in contributing! Please follow these steps:

## 1. Fork the Repository
Click **Fork** at the top-right to create your own copy.

## 2. Organize Your eBook
- Place your eBook in the correct **genre folder**.
- Create a **year subfolder** if it doesn't exist.
- File naming format: `Author - Book_Title.pdf`.

## 3. Submit a Pull Request
- Commit your changes with a clear message: `Add [Book Title] by [Author]`.
- Open a **Pull Request** (PR) to the main repo for review.

## 4. Rules
- Only share **non-copyrighted or freely distributable eBooks**.  
- Do **not upload scanned textbooks or copyrighted PDFs**.  
- Make sure files are under **100 MB** (or use Git LFS for larger files).

Thank you for contributing!
